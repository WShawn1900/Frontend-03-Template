# 一般命令语言的设计方式

##命令式语言的结构层级

1、Atom（语言最小组成单位）
    关键字、直接量、数字、true、变量名
    
2、Expression （表达式）
     Atom + 运算符 + 辅助符号
     
3、Statement（语句）
    Expression + 关键字 + 标识符等
    if 、for 、while 、
    
4、Structure (结构化) 用来帮助我们来组织代码、将代码分成不同的可复用块
    Function、class、Namespace、
    
5、Program (模块化)
    package、module、Library
